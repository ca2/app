# acme {#mainpage}

`acme` is the foundational, cross-platform layer of the
[ca2 framework](https://ca2.site/). It provides the core types, memory and
filesystem facilities, task infrastructure, platform services, and lightweight
user interaction primitives used by the layers above it.
